# Notes Plus

Notes Plus is a RuneLite external plugin scaffold for an in-game note workspace.

## Current status

This initial version adds a sidebar panel with placeholder UI so the plugin structure is ready for iterative development.

- Left side: folder tree with a single root folder (`My Notes`)
- Right side: text editor area
- Top actions: `New Folder`, `New Note`, `Rename`, `Delete`
- Drag and drop: not implemented yet
- Persistence/saving: not implemented yet

## Vision

Notes Plus is intended to evolve into a full note-management workflow inside RuneLite, including nested folders, note metadata, robust editing, and persisted storage while remaining Plugin Hub friendly.
